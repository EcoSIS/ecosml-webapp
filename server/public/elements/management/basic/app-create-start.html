Select Repository Type

You have the option to create a EcoSML Managed Repository or if you already have a GitHub repostory you can create a Registered Repository.
Recommended for users who are new to source code management (git) or don't want to manage their own repository. Managed repositories are easy to create and provide a simple interface for updating code. The repository will be hosted in the EcoSML GitHub organization.
Recommended for users who: have experince with git, want to host the repository in their own GitHub organization or already have a GitHub repository. You will provide EcoSML the url to your existing GitHub repository on the next screen.
